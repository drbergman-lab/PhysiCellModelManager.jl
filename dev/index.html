PhysiCellModelManager.jl

The PhysiCellModelManager.jl (PhysiCell Virtual Clinical Trials) package is a Julia package that provides a framework for running virtual clinical trials using PhysiCell simulations. See Getting started for getting PhysiCellModelManager.jl set up and running.
