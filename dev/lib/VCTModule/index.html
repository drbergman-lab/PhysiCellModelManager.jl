pcvct.readConstituentIDs — Method

readConstituentIDs(T::AbstractTrial)

Reads the constituent IDs for a given trial type T.

Arguments

  • T::AbstractTrial: An instance of a trial type.

Returns

  • A list of constituent IDs read from a CSV file.

Details

The function constructs a file path based on the type and ID of the trial T. It then reads the constituent IDs from a CSV file located at the constructed path.
