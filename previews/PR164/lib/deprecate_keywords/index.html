Deprecate Keywords

A light edit of the DeprecateKeywords.jl package to enable the deprecation warnings for users.

Public API

Private API

PhysiCellModelManager.DeprecateKeywords.@depkws — Macro
@depkws [force=false] def

Macro to deprecate keyword arguments. Use by wrapping a function signature, while using @deprecate(old_kw, new_kw) within the function signature to deprecate.

Examples

@depkws function f(; a=2, @deprecate(b, a))
    a
end
# force the deprecation warning to be emitted
@depkws force=true function f(; a=2, @deprecate(b, a))
    a
end
