Runner

Run simulations, monads, samplings, and trials in the ModelManager.jl framework.

Public API

Base.run — Method
run(T::AbstractTrial; force_recompile, kwargs...)

Run all pending simulations in T and return an MMOutput.

kwargs are forwarded to postSimulationProcessing for each completed simulation. For example, PhysiCellModelManager accepts prune_options.
